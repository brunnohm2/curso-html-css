Usando links externos

Utilizando a tag <a> para acessar outros links.

Atributo: href="" para on link do destino.

Atributo: target="" para o modo que a o link vai abrir.

  • _blank: vai abrir o link em uma nova janela em branco
  • _self: vai abrir o link na janela ou frame atual (padrão)
  • _parent: similar ao uso do _top em uma referência à janela mãe
  • _top: vai desfazer todos os frames e abrir o destino no navegador completo

Atributo: rel="" Existe um recurso bem interessante para links que é indicar qual é a natureza do destino usando o atributo rel. Esse atributo aceita vários valores, entre eles vou citar:

  • next: indica que o link é para a próxima parte do documento atual
  • prev: indica que o link é para a parte anterior do documento atual
  • author: indica que é um link para o site do autor do artigo atual
  • external: indica que é um link para outro site que não faz parte do site
  • nofollow: indica ao mecanismo de busca não seguir o link, usa-se em links de anúncios
